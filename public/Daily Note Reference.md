---
title: Daily Note Reference
---
This reference takes inspiration from the [Conventional
Commits](https://www.conventionalcommits.org/en/v1.0.0/) specification for Git
commit messages.

Rules:

* All entries MUST begin with one of the listed prefixes e.g. `thought:
  Build a note taking system`
* Prefixes MAY be appended with `!` to denote importance e.g. `todo!:
  Put the bins out`
* All references to dates MUST use the format `YYYY-MM-DD` e.g. `2000-01-02`

## General

| Prefix  | Description                                                           |
| ------- | --------------------------------------------------------------------- |
| log     | An entry recording an action, or event                                |
| media   | An entry linking to consumed media                                    |
| meta    | An entry reflecting on the note taking system itself                  |
| note    | An entry recording a event, feeling, or quick reflection from the day |
| thought | An entry capturing a fleeting thought, idea, or question              |
| todo    | An entry listing an actionable item, or task                          |

## Personal Life

| Prefix | Description                                                                      |
| ------ | -------------------------------------------------------------------------------- |
| food   | An entry logging notable meals, food, or drink                                   |
| habit  | An entry reflecting on recurring behaviours                                      |
| health | An entry recording state of physical, emotional, or mental health, and self-care |

## Work & Projects

| Prefix | Description                                           |
| ------ | ----------------------------------------------------- |
| start  | An entry marking the start of a task, or project      |
| done   | An entry marking the completion of a task, or project |
